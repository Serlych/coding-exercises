Requirements:

1. Schedule a tour on a property
2. Rescheduling should be available
3. Cancelling tours should be available
4. One user must perform the invite
5. The agent defines the business hours and days
6. Only one user per time-range
7. Store the booking status
8. Show only three days ahead

Constraints:

1. User and can potentially invite some over
2. Not timezone aware
3. Need to schedule on business hours
4. Send an error if the user sends a tour request outside business hours
5. No need for notifications before or after the tour
6. The duration of the appointments belongs to the agent
7. 10 minute buffer for back to back tours
8. No follow-ups
9. No need to track if it was rescheduled
